Lightboxes

Colorbox

You can use Justified Gallery with any lightbox in the way you prefer. In this tutorial you can see how to use Colorbox and Swipebox, but the same principles can be applied to use any other lightbox.

To use Colorbox you need to create a listener for the jg.complete callback to initialize Colorbox. Note that we are also using the rel = 'gallery2' option, that sets to 'gallery1' the rel attribute for all the links. In this way Colorbox can create a group of images.

{% highlight html %}
...
{% endhighlight %} {% highlight javascript %} $('#colorboxExample').justifiedGallery({ lastRow : 'nojustify', rowHeight : 100, rel : 'gallery1', //replace with 'gallery1' the rel attribute of each link margins : 1 }).on('jg.complete', function () { $(this).find('a').colorbox({ maxWidth : '80%', maxHeight : '80%', opacity : 0.8, transition : 'elastic', current : '' }); }); {% endhighlight %}

Swipebox

To use Swipebox you need to use again a listener for the jg.complete callback. Unfortunately, due to a Swipebox's misteriousely bug, the simple code:

{% highlight javascript %} $(this).find('a').swipebox(); {% endhighlight %}

doesn't work properly. In this way Swipebox activate itself for all photos in the same page. To fix that, one needs to select the gallery using a complete selector like the following:

{% highlight javascript %} $('#swipeboxExample a').swipebox(); {% endhighlight %}

Look at the following code for a complete example:

{% highlight html %}
...
{% endhighlight %} {% highlight javascript %} $('#swipeboxExample').justifiedGallery({ lastRow : 'nojustify', rowHeight : 100, rel : 'gallery2', margins : 1 }).on('jg.complete', function () { $('#swipeboxExample a').swipebox(); }); {% endhighlight %}

Multiple gallery with one call Colorbox

The three galleries are created with a single call to the plugin. Note that, to create the Colorbox groups, the images have different rel tags.

{% highlight html %}
...
...
...
{% endhighlight %} {% highlight javascript %} $('.myExMul').justifiedGallery({ rowHeight : 50 }).on('jg.complete', function () { $(this).find('a').colorbox({ maxWidth : '80%', maxHeight : '80%', opacity : 0.8, transition : 'elastic', current : '' }); }); {% endhighlight %}

To set the rel automatically, just call the Justified Gallery inside a foreach.

{% highlight javascript %} $('.myExMul').each(function (i, el) { $(el).justifiedGallery({rel: 'myExMul-' + i}).on('jg.complete', function () { $(this).find('a').colorbox({ maxWidth : '80%', maxHeight : '80%', opacity : 0.8, transition : 'elastic', current : '' }); }); }); {% endhighlight %}

Multiple gallery with one call swipebox

To use the Swipebox it's a little hard again due to the same bug. You need to call it once, and to select all the images in one selector.

{% highlight javascript %} $('.myExMul').each(function (i, el) { $(el).justifiedGallery({rel: 'myExMul-' + i}).on('jg.complete', function () { if (i == 1) $('.myExMul a').swipebox(); }); }); {% endhighlight %}
